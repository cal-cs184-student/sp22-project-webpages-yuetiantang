Part 2. Simulation via numerical integration

Describe the effects of changing the spring constant ks; how does the cloth behave from start to rest with a very low ks? A high ks?

·       Changing the spring constant ks makes the cloth tougher or softer. With a larger ks, springs inside require greater forces to be stretched. A smaller ks makes a spring easier to be stretched.

·       When I set ks to a very low value (ks=1e-05), the cloth looks fine and behaves without any abnormality. It just seems to be slightly stretched on y-axis. The reason is that we enforced a 10% maximum stretch length, and the springs cannot be stretched any further beyond that amount.

·       When I set ks to a very large value (ks=300000), the cloth looks fine at the beginning, then it starts shrinking and wrinkling on its own. It keeps moving and trembling.

What about for density?

·       When I set density to a very small value (density=0.2), the cloth looks fine at the beginning, then it starts shrinking and wrinkling on its own. It keeps moving and trembling.

·       When I set density to a very large value (density=1e+10), the cloth looks fine.

What about for damping?

·       When I set damping to a very small value (damping=0.011494), the cloth swings for approximately 45 seconds before comes to rest. It won’t rest when damping is set to 0.

·       When I set damping to a very large value (damping=1), the cloth slowly sags and comes to a complete rest in a short period of time (15 seconds).
